Homeroom is a video conference application, enhanced by AI features for facial recognition and hand detection.
This is the client desktop application.


## Technologies

- Electron
- React
- Tailwind
- Firebase client


## Getting Started

Homeroom is a React SPA, and we use Electron to simulate the server as a desktop application.
As such, you are able to run Homeroom both as a web application or as a desktop application.

1. Clone repo
2. `npm install`
3. `npm run dev`
4. Electron window will pop up automatically, optionally you can also browse to `http://localhost:3000`
